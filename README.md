# domina
Domina افزونه‌ای حرفه‌ای برای وردپرس است که امکان نمایش صفحات تعمیرات، به‌روزرسانی، در دست ساخت و معرفی یا فروش دامنه را فراهم می‌کند. این افزونه با قالب‌های واکنش‌گرا، پشتیبانی کامل از فارسی و RTL، تنظیمات امنیتی، فرم تماس امن، کنترل دسترسی مدیران، پاسخ HTTP 503 و شخصی‌سازی لوگو، رنگ‌ها و محتوای صفحه، تجربه‌ای سریع و حرفه‌ای ارائه می‌دهد.
